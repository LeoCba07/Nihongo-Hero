## The Problem

Traditional Japanese study apps are boring. Flashcards work, but they don't make you *want* to come back.

## The Solution

Nihongo Hero transforms language study into an RPG. Battle enemies by answering vocabulary questions ‚Äî correct answers deal damage, mistakes cost health. Learning through combat.

### Features

- **Turn-based combat** ‚Äî answer Japanese questions to attack enemies
- **TTS pronunciation** ‚Äî hear questions spoken via VoiceVox API
- **Dynamic animations** ‚Äî sword strikes, screen shake, floating damage numbers
- **Story mode** ‚Äî progress through themed levels with increasing difficulty
- **Mobile-first PWA** ‚Äî installable app for learning on the go
- **Leaderboard** ‚Äî compete with other learners
